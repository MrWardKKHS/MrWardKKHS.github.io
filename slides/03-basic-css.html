CSS Structure

CSS follows a simple format:

  property: value;
}

  color: orange;
}

CSS Selectors Overview

  • tagname – Styles all tags (e.g., h1)
  • .classname – Styles elements with a class
  • #idname – Styles a specific element by ID

Example: ID and Class Together


<p id="special" class="highlight">
  This paragraph has both an ID and a class.
</p>

This paragraph has both an ID and a class.

Example: Styling the body


body {
  color: red;
}

This sets the default text color for the whole page to red.

This paragraph inherits red text from body.

Example: Styling a Tag


h1 {
  color: #00ff00;
}

This changes the color of all <h1> elements to bright green.

This is a green heading

Example: Styling a Class


p.ex {
  color: blue;
}

This styles paragraphs with class="ex" differently.

This paragraph is blue because it has the class "ex".

Example: Overriding with an ID


#special {
  color: purple;
}

IDs are stronger than classes or tags, so they override the others

This paragraph is purple, not blue!

CSS Properties We'll Use Today

  • color – text color
  • background-color – background shade
  • padding – inside space
  • margin – outside space
  • border – border around things
  • border-radius – rounded corners
  • max-width – control size of elements
  • text-align – left, right, center text
