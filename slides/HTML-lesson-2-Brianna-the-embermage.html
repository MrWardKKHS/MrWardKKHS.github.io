Tags Revision Quiz

Which tag is used for the main heading on a webpage?

<h1>Main heading</h1>

What does the href attribute do inside an <a> tag?

href tells the browser where the link should go.

<a href="https://example.com">Visit Site</a>

Hypertext REFrence

Which tag is used to add an image to your page?

<img src="path-to-image.jpg" alt="Description">

How would you correctly code a simple bullet list?

<ul>
  <li>First point</li>
  <li>Second point</li>
</ul>

What mistakes do you see here?

<a>Click here<a>

It‚Äôs missing the href="" attribute.

The end tag was not a closing tag.

Correct version:

<a href="https://example.com">Click here</a>

Which tag would you use for a sub-heading under an <h1>?

Use an <h2> tag for sub-headings under <h1>.

<h2>Sub-heading</h2>

What is wrong with this hyperlink?

<a href=https://example.com>Click here</a>

The URL must be inside quotes ("").

Correct:

<a href="https://example.com">Click here</a>

True or False: You can nest a list inside a list?

True! Example:

<ul>
  <li>Main Point
    <ul>
      <li>Sub-point</li>
      <li>Sub-point 2</li>
      <li>Sub-point 3</li>
      <li>Sub-point 4</li>
    </ul>
  </li>
  <li>Second Main Point
</ul>
