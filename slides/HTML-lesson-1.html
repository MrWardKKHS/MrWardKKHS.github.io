Learning Outcome 1

✅ Describe what HTML is and what it’s used for

HTML stands for HyperText Markup Language. It's the standard language used to create and structure content on the web. Every webpage you’ve ever used is powered by HTML.

Basic HTML Example


            <!DOCTYPE html>
            <html>
              <head>
                <title>My Webpage</title>
              </head>
              <body>
                <h1>Welcome!</h1>
                <p>This is a simple HTML page.</p>
              </body>
            </html>

The hyper in hypertext is links

<a href="https://www.rhs.school.nz">RJHS</a>

Learning Outcome 2

✅ Identify the role of tags in structuring a webpage

What is a tag?

Tags wrap around inner text to give it meaning.

<p>This is a paragraph.</p>

This tells the browser: “Display this as a paragraph.”

Common HTML Tags

TagUseExample
 
<h1>
Main heading<h1>Title</h1>
<p>
Paragraph<p>Text</p>
<ul><li>
Bullet list<ul><li>Item</li></ul>
<a>
Link<a href="#">Click me</a>
<img>
Image<img src="..." alt="...">

Nesting Tags

Tags can go inside other tags. For example:

<p>We support <strong>apprentice rights</strong> for all.</p>

This makes “apprentice rights” bold inside the paragraph.

Rendered Output

Example code:

<h2>My Campaign</h2>
<p>We believe in <em>fairness</em> and <strong>free capes</strong>.</p>

It renders like:

My Campaign

We believe in fairness and free capes.
